# Inheritance UMaT Management System

A simple Python Object-Oriented Programming (OOP) system designed to manage different types of personnel on the UMaT campus using class inheritance.

## Task Overview
- **Parent Class:** `Person` containing common attributes (`name`, `age`) and method (`display_info()`).
- **Child Classes:** - `Student` adds `student_id` and `enroll_course()`.
  - `Lecturer` adds `employee_id` and `teach_course()`.
- Demonstrates constructor chaining using `super()` and method overriding.
